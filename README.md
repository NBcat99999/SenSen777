# FutureFlow CFO 工作中台

面向初创团队的本地经营与融资管理系统。默认只保留经营总览、报销票据、代账交接和融资雷达四个入口。

- 管理层首页：现金、月度经常性收入、客户数、收支、现金可支撑时间和待回款。
- CFO 规则罗盘：现金安全、增长验证、单位经济、财务合规和融资准备度五维评分。
- 专业指标中心：MRR（月度经常性收入）、ARR（年度经常性收入）、NRR（净收入留存率）、CAC（客户获取成本）、LTV（客户终身价值）等指标及词典。
- 客户模型：初始为空，可手工录入合同、月度预算、毛利率、获客成本、账期、续约概率及客户状态，并联动收入、毛利、现金预测、罗盘和融资评分。
- 报销票据：录入员工报销并留存附件哈希值。
- 全局联动：待审核报销降低可用现金；已审核报销计入费用和应付；已支付报销减少实际现金；已驳回记录退出经营计算。
- 代账交接：按月导出资金流水、员工报销、票据索引及原始附件压缩包。
- 融资雷达：机构候选库及近期公开融资信号；候选匹配不代表机构已表达投资意向。
- 国内融资筛选：按数据、大交通、旅游、新兴入境、出境出海和电商关键词筛选国内机构、产业资本及政府产业基金。

核心计算使用确定性公式，除主动点击融资雷达更新外，不调用外部服务。

界面采用高对比度亮色管理风格，首页默认仅展示六个核心数字及三项管理动作，专业指标和明细按需展开。

客户与合同页面支持新增合同，并可直接修改月度预算、毛利率、获客成本、账期、续约概率和客户状态。修改后自动联动 MRR（月度经常性收入）、ARR（年度经常性收入）、预计毛利、现金预测、单位经济、CFO 规则罗盘及融资机构匹配分。实际现金只在真实回款或付款发生后变化。

## 数据保护

- 每次启动前自动备份完整数据目录及票据附件。
- 每次写入前自动保存旧状态并生成 SHA-256（安全哈希值）。
- 功能升级只增加缺失字段，不覆盖或删除旧字段与旧记录。
- 导入采用非破坏性合并，不整体替换当前数据。
- 详细规则见 `DATA_PROTECTION.md`。

## 启动

在 macOS（苹果电脑操作系统）中双击 `打开FutureFlow_CFO工作中台.command`，或在项目目录运行：

```bash
python3 server.py --port 8876
```

访问 `http://127.0.0.1:8876`。

## 数据

- 核心数据：`data/state.json`
- 证据文件：`data/evidence/`
- 自动周报索引：核心数据中的 `weeklyReports`
- 完整备份：在“设置备份”模块下载

核心经营数据、票据附件及备份默认不纳入 Git（版本控制系统），克隆仓库后首次启动会自动创建本机数据文件。

仓库版本不包含演示客户、样例交易、样例往来款或任何本机经营记录。

## 每周更新

服务器启动时会检查本周报告；若不存在则自动生成。服务器持续运行时每小时检查一次，因此每周一会自动生成新的周报。
